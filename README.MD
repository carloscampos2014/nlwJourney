# Projeto NLW Journey

Esse projeto foi desenvolvido na semana NLW Journey da Rocketseat

## Descrição
 O projeto trate-se de uma API para cadastrar viagens e atividades de viagens

## Desenvolvimento
O projeto foi desenvolvido usando Net Core 8, Entity FrameWork, SqLite, esta dividido nas seguintes camadas:

 - API
 - Application
 - Communication
 - Exceptions
 - Infrastructure
